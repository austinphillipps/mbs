# mbsmanager (Flutter + Firebase)

Ce dépôt contient désormais une application Flutter (`flutter_app/`) qui reprend les écrans clés (dashboard, stocks, clients, commandes, fournisseurs, analyses, profil) en utilisant Firebase pour l'authentification et les données (Firestore).

Consultez `flutter_app/README.md` pour la configuration détaillée (installation Flutter, configuration Firebase via `flutterfire configure` et collections attendues).
